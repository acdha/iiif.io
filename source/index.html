Why should I Use IIIF? How do I Get Started?

Community Driven

IIIF is driven by a consortium of libraries, museums, universities, companies, and image repositories committed to opening up access to images from every culture.

Defined APIs

IIIF is anchored by well defined application programming interfaces, developed against real use cases and vetted by an open and rigorous community process.

Plug ’n’ Play Software

IIIF encourages and supports the development of interoperable image serving and viewing software that is easy to install and provides a dazzling user experience.
